HTML - Lists

HTML lists are used to organize content in a structured and meaningful way. There are three main types of lists in HTML: ordered lists (<ol>), unordered lists (<ul>), and definition lists (<dl>).

Ordered lists (<ol>)

An ordered list is used when the order of items is important. Each item in an ordered list is numbered automatically.

Example:

Output:

  1. First Item
  2. Second Item
  3. Third Item

Customizing Ordered Lists

You can customize the numbering style of ordered lists using the type attribute.

  • Numerical (default): <ol type="1"> (1, 2, 3...)
  • Uppercase Letters: <ol type="A"> (A, B, C...)
  • Lowercase Letters: <ol type="a"> (a, b, c...)
  • Uppercase Roman Numerals: <ol type="I"> (I, II, III...)
  • Lowercase Roman Numerals: <ol type="i"> (i, ii, iii...)

Example:

Output:

  1. First Item
  2. Second Item
  3. Third Item

Unordered Lists (<ul>)

An unordered list is used when the order of items is not important. Each item is marked with a bullet point by default.

Example:

Output:

  • First Item
  • Second Item
  • Third Item

Description Lists (<dl>)

A description list is used for terms and their corresponding definitions. It consists of pairs of terms (<dt>) and descriptions (<dd>).

Example:

Output:

HTML
A markup language used to create web pages.
CSS
A stylesheet language used to describe the presentation of a web page.
JavaScript
A programming language used to add interactivity to a web page.

Nested Lists

You can also create nested lists, where one list is placed inside another. This is useful for creating multi-level categories.

Example Nested Ordered Lists

Output:

  1. Item 1
    1. Subitem 1
    2. Subitem 2
  2. Item 2
    1. Subitem 3
    2. Subitem 4

Example Unordered Lists

Output:

  • Item 1
    • Subitem 1
    • Subitem 2
  • Item 2
    • Subitem 3
    • Subitem 4
