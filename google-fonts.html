Google Fonts

1. Navigate to the Google Fonts site here.

2. Select the font(s) that you like, then click on "Select Regular 400" (other options might be available).

3. Once a font has been added. The side window will appear containing 2 sets of code (embedding the font & CSS rules to specify families).

note: You can embed the Google Font either by adding the <link> code in your HTML page OR the @import in your CSS

3.1 Option 1: Embedding Google Fonts on HTML

Add the code from <link> window above the closing head element


3.2 Option 2: Embedding Google Fonts in CSS

Add the code from @import on the top of your stylesheet

4. Add the 2nd part of the code to your text block
