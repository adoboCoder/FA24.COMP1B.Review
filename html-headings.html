HTML - Headings

Headings are an essential part of structuring web content. They help organize the content and improve accessibility and SEO (Search Engine Optimization). In HTML, headings range from <h1> to <h6>, where <h1> represents the highest or most important heading level, and <h6> represents the lowest.

What are HTML Headings?

Headings in HTML define the titles or subtitles of sections on a webpage. They provide a way to organize and structure content for users and search engines. HTML uses six levels of headings, where:

  • <h1> is the main heading (used for page or section titles)
  • <h2> to <h6> are subheadings, with <h6> being the least important.

Examples of Headings:


Output:

This is an H1 Heading

This is an H2 Heading

This is an H3 Heading

Used as a sub-subheading under the H2 heading.

This is an H4 Heading

Used for further sub-dividing content.

This is an H5 Heading

Less common, used for minor divisions.

This is an H6 Heading

Rarely used, often for the smallest sections.
